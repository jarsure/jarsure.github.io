Java接口中的静态方法和默认方法

28 7月 2021 in java

Java8新增了接口的默认方法和静态方法

Java9新增了接口的私有方法

默认方法–java8

写在接口内部,

可以拥有自己的方法体,

当接口被实现时,有两种选择

1.重写该方法

2.不重写

​ 则继承父类的方法

当一个类实现了多个接口,这些接口中有相同的默认方法时,

必须重写该方法

静态方法(接口中)–java8

写在接口中时,只能通过接口名调用.

不可以通过实现类的类名或者实现类的对象调用

私有方法–java9

private修饰,

必须拥有方法体,

只在接口内被调用,

该方法只在接口内有用,无法被子类实现或继承
