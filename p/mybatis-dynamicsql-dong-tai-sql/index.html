        select * from t_emp where 1=1
        <if test="empName != null and empName != ''">
            and emp_name = #{empName}

where

where标签一般和if标签同时使用

1.如果where标签中有条件成立,会自动生成where关键字

2.会自动将where标签中内容前多余的and去掉,内容后的无法去除

3.若where标签中没有一个条件成立,则where标签没有任何功能

trim

prefix,suffix: 在标签内全部内容的前边或后边添加指定内容.

prefixOverrides,suffixOverrides:在标签内全部内容的前边或后边去掉指定内容.

        select * from t_emp
        <trim prefix="where" suffixOverrides="and">
            <if test="empName != null and empName != ''">
                emp_name = #{empName} and
            </if>
            <if test="age != null and age != ''">
                age = #{age} and
            </if>
            <if test="gender != null and gender != ''">
                gender = #{gender} and
            </if>
        </trim>
    </select>

<!--List<Emp> getEmpByChoose(Emp emp);-->
    <select id="getEmpByChoose" resultType="emp">
        select * from t_emp
        <where>
            <choose>
                <when test="empName != null and empName != ''">
                    emp_name =  #{empName}
                </when>
                <when test="age != null and age != ''">
                    age =  #{age}
                </when>
                <when test="gender != null and gender != ''">
                    gender =  #{gender}
                </when>
            </choose>
        </where>
    </select>

通过任意一条when的条件,则不会继续判定之后的when条件.

<!--void deleteMoreEmp(@Param("empIds") Integer[] empIds);-->
